Q1: What is HTML?
HTML stands for HyperText Markup Language.
Q2: What is the difference between ID and Class in CSS?
An ID is a unique identifier used to style a single element, while a class can be used to apply styles to multiple elements. ID uses `#` in CSS, whereas class uses `.`.
Q3: What is the purpose of CSS?
CSS (Cascading Style Sheets) is used to style HTML elements.
